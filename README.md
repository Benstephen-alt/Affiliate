# StakersPro Affiliate Frontend V2

## Routes

- `/` registration and login page
- `/dashboard` protected affiliate dashboard
- `/admin` admin dashboard

## Run Locally

```bash
npm install
cp .env.example .env
npm run dev
```

Set `VITE_API_BASE` to your backend URL in production.

## UI Updates

- Bright modern interface.
- Register/Login links are not shown on the dashboard.
- User must log in again after logout or expired session.
- Code files are formatted with readable line breaks.
